# ✅ Angular Mastery Checklist

## 1. Débutant – "Je connais Angular"
- [ ] **Installation et projet de base (Angular CLI)**  
  → Créer un projet avec `ng new`, lancer avec `ng serve`.
- [ ] **Composants**  
  → Comprendre `@Component`, créer un composant avec template et style.
- [ ] **Templates & Data Binding**  
  → Savoir utiliser `{{ }}`, `[property]`, `(event)`, `[(ngModel)]`.
- [ ] **Directives**  
  → Utiliser `*ngIf`, `*ngFor`, `ngClass`, `ngStyle`.

## 2. Intermédiaire – "Je suis à l’aise"
- [ ] **Services & Dependency Injection**  
  → Créer un service avec `@Injectable`, l’injecter dans des composants.
- [ ] **Routing**  
  → Définir des routes avec `RouterModule`, utiliser `routerLink`.
- [ ] **Formulaires**  
  → Maîtriser Template-driven et Reactive Forms, validation (`Validators`).
- [ ] **HTTP Client**  
  → Faire des requêtes avec `HttpClient` (`get`, `post`, interceptors`).
- [ ] **Modules**  
  → Comprendre `NgModule`, organiser un projet avec des modules partagés/feature.

## 3. Avancé – "Je maîtrise Angular"
- [ ] **RxJS et Observables**  
  → Utiliser `map`, `filter`, `switchMap`, `mergeMap`, `Subject`.
- [ ] **Lifecycle Hooks**  
  → Maîtriser `ngOnInit`, `ngOnDestroy`, `ngAfterViewInit`.
- [ ] **Change Detection**  
  → Comprendre le fonctionnement, `OnPush` strategy, `async` pipe.
- [ ] **Lazy Loading**  
  → Charger des modules à la demande pour optimiser les performances.
- [ ] **Testing**  
  → Écrire des tests unitaires (Jasmine, Karma) et end-to-end (Protractor, Cypress).

## 4. Expert – "Je domine Angular"
- [ ] **State Management**  
  → Maîtriser NgRx/Akita pour gérer le state global.
- [ ] **Performance & optimisation**  
  → Préchargement, cache, bundle optimisé (`ng build --prod`).
- [ ] **Architecture avancée**  
  → Organiser en Clean Architecture, Domain Driven Design.
- [ ] **SSR et Universal**  
  → Gérer le rendu côté serveur avec Angular Universal.
- [ ] **Contributions & libs**  
  → Écrire une bibliothèque Angular réutilisable, contribuer à des projets open-source Angular.
